‚ú® Additions & Improvements
  • WebRTC Live Spectate - Watch players in real-time through a live stream of their screen.

  • Expanded Logging
    • Item spawning / giving (self + others)

  • Optimized Assets - Player map skins converted to WEBP and unused files removed.

  • Report Notification Sound - A sound now plays when a new report is received.
  • Player Report Chat Flow - Report messages from staff no longer auto-focus the chat. Players are now prompted to press END to manually open and focus the report chat.
  • Analytics Update - The Tebex stats page has been moved into the Analytics page.
  • Install Status Rewrite - Now determined dynamically based on database tables + at least one superadmin (no longer stored in server KVP).
  • Editable Server Name - SuperAdmin users can now edit the server name directly from the dashboard, no longer requiring a re-install.
